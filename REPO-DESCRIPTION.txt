=== ABOUT SECTION (Extended Description) ===

A complete full-stack printer business website that allows customers to upload files, select print options, calculate pricing, process payments via Razorpay, and print documents directly to Raspberry Pi-connected printers.

Key Features:
- Modern React frontend with TypeScript and Tailwind CSS
- Node.js/Express backend with MongoDB database
- Razorpay payment gateway integration
- File upload (PDF, DOC, DOCX) with validation
- Real-time pricing calculator with bulk discounts
- Order management and tracking system
- Raspberry Pi printer integration via Cloudflare Tunnel
- Responsive design for mobile and desktop

Tech Stack:
- Frontend: React 18, TypeScript, Tailwind CSS, Radix UI
- Backend: Node.js, Express.js, MongoDB, Mongoose
- Payment: Razorpay SDK
- Printer: Raspberry Pi, CUPS, Cloudflare Tunnel
